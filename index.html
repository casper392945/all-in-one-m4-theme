How is It Made

A short description of the assembly process

1

Search and Read the List of Purchased M4 Themes

Open to view the file ...\addons\list.json. Find all "M4" and "theme". Save the search results in a separate file of the list of installed M4 themes.

2

Make New Theme in Addons Folder

Make new folder in ...\addons, for example: "My Super Theme".

3

Copy to New Theme All M4 Themes Projects

Go through the list of M4 themes and copy all project directories to the "My Super Theme" folder.

4

Copy to New Theme All M4 Themes Fonts

Go through the list of M4 themes and copy all fonts directories to the "My Super Theme" folder.

5

Copy to New Theme All M4 Themes Plugins

Go through the list of M4 themes and copy all plugins directories to the "My Super Theme" folder.

6

Copy to New Theme All M4 Themes Components

Go through the list of M4 themes and copy all components directories to the "My Super Theme" folder.

7

Make params.json for "My Super Theme"

Copy the file params.json from any M4 theme. Open params.json in editor. Replace the name and title to "My Super Theme". Add copied components. Add copied fonts.

8

That's all

p.s. The work was performed manually using the file manager and text editor. When editing params.json, no block was harmed 😀 since they were not edited, just only copied as is.
